#AMD

正如上周所预期的，价格从看涨的 POI 反弹至 91.58。这种反应还不够强烈，不足以打破市场结构的上行趋势。从这里开始，我预计看涨 POI 会进一步缓解，或者价格会开始下跌。

#NFLX
