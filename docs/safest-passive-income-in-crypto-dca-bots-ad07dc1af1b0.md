# **什么是 DCA(美元成本平均法)？🌈**

1.  美元成本平均法是一种在固定的时间间隔内系统地投资大量资金的做法，不考虑证券的价格⏱
2.  美元成本平均法可以减少价格波动的总体影响，降低每股平均成本💵
3.  它消除了市场时机的陷阱，比如只在价格已经上涨时买入📈
4.  它让你的投资不带感情色彩，防止你潜在地损害你的投资组合的回报🥹

DCA 已经存在了几十年，无数传统投资者使用这种技术，尤其是在熊市期间。这是一种简单而古老的东西，但却非常有效。
