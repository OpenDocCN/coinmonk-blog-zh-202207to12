# 真正的美元加密货币为何诞生？

*   交易商和交易所——真正的美元(TUSD)加密货币可以被认为是一种伟大的工具，用于对冲市场波动，并进入市场，而无需首先购买 ETH/BTC。ETH 代表以太坊，BTC 代表比特币货币。
