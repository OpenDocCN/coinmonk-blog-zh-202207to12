我们生活在一个时髦词汇的世界里:人工智能、数据科学、区块链、web3、量子计算、智能合约、游戏化……人们很容易感到困惑，不会关注真正重要的东西，但检查一下其他领域也无妨。

区块链和智能合约是一些新的领域，但有很多资源可以帮助我们开始。

1.区块链和以太坊基础知识

虽然比特币是加密货币的主流话题，但如果你希望获得更多关于区块链和智能合约的知识，那么ETH(以太坊的缩写)是一条出路。

  1. 编码技术讲解区块链技术(2小时视频)
  2. 《区块链加密:你需要知道的一切》作者101区块链
  3. 掌握以太坊书
  4. 以太坊开发者资源

2.智能合同的第一步

智能合约是开发者用编程语言写的，最常用的是Solidity，不过欢迎你去查 Vyper (特别是如果你熟悉Python的话)。

  1. 坚固性文件
  2. 智能合同工程师
  3. 隐型僵尸(一种游戏化的学习方式)
  4. 举例说明的可靠性
  5. 安全实体101
  6. Secureum Solidity 201
  7. 固体气体优化列表
  8. 捕捉以太 (CTF式学习)
