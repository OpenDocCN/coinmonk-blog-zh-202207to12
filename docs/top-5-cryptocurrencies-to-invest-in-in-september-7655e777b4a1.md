# 卡尔达诺

Cardano (ADA)是一种加密货币，由包括工程师、数学家和密码学专家在内的研究人员开发。这是一个使用大毒蛇共识机制的利害关系证明系统。查尔斯·霍金森也是以太坊最初的五位创始人之一，他被认为是该项目的发起人。
