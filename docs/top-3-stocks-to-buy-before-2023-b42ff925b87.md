# 力拓公司

总部位于英国伦敦的矿业集团力拓(纽约证券交易所代码:Rio)自称为“世界领先的矿业集团”。力拓目前的市值接近 1180 亿美元。自年初以来，约 8%的初始投资已经返还给了力拓股东。几乎所有这些活力都发生在最近一个月，股价上涨了 20%。
