银行是金融机构系统性风险最明显的代表。当一家或多家金融机构破产或倒闭时，金融体系就会变得不那么安全:人们再次无法获得传统的银行服务。这一次对单个机构的后果也不同:如果出现银行挤兑，将导致消费者和其他金融机构的成本上升。
