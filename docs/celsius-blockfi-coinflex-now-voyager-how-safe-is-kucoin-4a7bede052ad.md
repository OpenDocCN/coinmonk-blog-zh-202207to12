# 传染是真实的，但 FUD 可能不是……或者可能是？

因此，在 twitter 和 discord 上关注许多不同的 CeFi 和 crypto 交换平台后，传染病肯定正在发生，或者至少围绕传染病的 FUD 肯定正在发生。也许每个平台现在都被要求提高透明度，人们会问更多关于如何/是否利用杠杆、资金是否涉及算法稳定性的问题，或者换句话说，基本上是平台的收益是如何产生的，以及这些方法如何获得高收益。
