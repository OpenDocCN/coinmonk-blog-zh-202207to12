# 链环(链环)

Chainlink 是一个 oracle 网络，它将智能合约与现实世界的数据联系起来。它通过允许访问支付、事件等，弥合了智能合约与世界其他地方之间的差距。它也不会以任何方式危及网络的安全性和可靠性。
