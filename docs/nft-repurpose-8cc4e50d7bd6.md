NFT 或不可替代的令牌是一个数字产权！由于任何数字创作都可以复制，NFT 会放上一个独特的数字签名来认领你的物品。

但是，人们开始滥用他们的权力来利用他人，包括欺负他人。

拥有 NFT 可能只会授予您特定的数字所有权。除了证明拥有 NFTs 的权限外，还必须验证拥有 NFTs 的权限
