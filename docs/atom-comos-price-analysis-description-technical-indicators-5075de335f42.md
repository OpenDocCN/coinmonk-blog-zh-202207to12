**什么是宇宙？**

Cosmos 是一个独立的区块链网络，它使用 IBC(一种基于 TCP/IP 的互操作性协议)安全地传输价值和数据。Cosmos Hub，通常被称为“Gaia”，是一个利益相关链，它使用本地硬币原子作为 Cosmos 网络中区块链之间 IBC 数据包路由的枢纽。拜占庭容错(BFT)利害关系证明共识机制 Tendermint 保护宇宙中心以及宇宙网络中的大部分区块链。
