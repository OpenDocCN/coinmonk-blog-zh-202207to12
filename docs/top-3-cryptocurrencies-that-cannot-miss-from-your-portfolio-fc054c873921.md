# 比特币(BTC)

中本聪创造了比特币，一种数字货币和支付系统。网络中的节点使用密码术来验证交易，然后交易被添加到被称为区块链的公共分布式账本中。比特币的特殊之处在于，它永远只有 2100 万枚。
