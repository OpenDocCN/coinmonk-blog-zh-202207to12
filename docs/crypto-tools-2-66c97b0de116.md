# #1 南森. ai

![](img/e740048a3fc27cd0792e41a7f14e4539.png)

Source: Nansen.ai

[Nansen](https://www.nansen.ai/) 是一个强大的区块链分析平台，结合了链上数据和来自数百万个钱包标签的海量数据。它是最强大的链上分析平台之一，由一些世界领先的风投(a16Z，比特币基地风险投资，潘迪拉，…)支持。).
