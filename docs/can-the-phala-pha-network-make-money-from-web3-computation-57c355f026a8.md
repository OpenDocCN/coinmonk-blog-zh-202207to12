# Phala (PHA)网络能从 Web3 计算中赚钱吗？

**Phala Network (PHA)** 是一种隐私增强的无权限计算协议。

解释一下，Phala 通过只在硬件安全区域处理信息来保护敏感数据。他们声称，Phala 通过防篡改和不可信的执行来保护数据免受系统级攻击。因此，Phala 声称他们保证了代码和数据的完整性，因为协议只允许授权行为。
