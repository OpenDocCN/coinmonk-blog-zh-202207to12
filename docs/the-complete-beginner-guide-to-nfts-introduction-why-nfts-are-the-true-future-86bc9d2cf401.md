# Brefly，什么是 NFT？

NFT 代表不可替代令牌。“可替代”是什么意思？简单的答案是可替代意味着容易被替代。所以，不可替代意味着它不能被替代。

“代币”一词表示区块链所有权的证明。

# 我为什么要为我可以免费点击并保存的东西付费呢？

这是一个很好的问题，可以用一个简单的类比来回答。获得蒙娜丽莎的复制品很简单，但最终，你并不拥有蒙娜丽莎；你拥有一幅蒙娜丽莎的复制品。类似地，当你右击并保存数字艺术时，你有一个拷贝。你并不拥有原件。
