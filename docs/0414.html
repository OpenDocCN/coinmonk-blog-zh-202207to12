Windows 11更新复制了流行的Android功能，我们非常高兴

原文：https://medium.com/coinmonks/windows-11-update-copies-popular-android-feature-and-we-couldnt-be-happier-21a28f8a685f?source=collection_archive---------32-----------------------#2022-07-08

在之前的版本中被取消后，微软重新设计了Windows 11 的“打开方式”对话框，让你可以在不同于默认设置的应用程序中打开文件。

这一新设计最终消除了自 Windows 8 以来围绕消息框的混乱，对话框无法清楚地表明文件是否会在未来用新的默认应用程序打开，或者只是那一次。
