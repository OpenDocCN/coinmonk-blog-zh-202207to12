# **什么是 Monero？**

Monero 是一个字节码分支，通过环签名、秘密地址、私人交易和 bulletproofs 提供匿名值传输，重点是隐私保护和可替代性。与其竞争对手可选的隐私保护功能相反，Monero 默认提供匿名功能。Monero 的工作证明算法不断发展，以保持分散化，并挫败 ASIC 的主导地位和采矿流程集中化。
