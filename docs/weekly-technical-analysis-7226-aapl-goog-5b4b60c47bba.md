7226 美元

价格已经连续几个月保持稳定。价格最近在上方跳空，并在 1540 点获得流动性，但无法收于上方。从那以后，价格一直呈下降趋势。现在，我们看到价格建立了卖方流动性。这可能是一个短期看跌目标，价格可能会进一步向 1242 点的低点移动。但首先，在我们看到任何下跌反应之前，价格更有可能填补 1481 点的公允价值缺口。
