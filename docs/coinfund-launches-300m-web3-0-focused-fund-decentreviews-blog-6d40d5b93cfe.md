## 快速拍摄:

*   CoinFund 为 Web3 品牌推出 3 亿美元的基金
*   该基金还将帮助一些现有的 CoinFund 支持的品牌从种子期发展到首轮融资
*   这延续了 CoinFund 投资早期品牌的历史，包括 Solana、Rarible 和 Moonbeam
