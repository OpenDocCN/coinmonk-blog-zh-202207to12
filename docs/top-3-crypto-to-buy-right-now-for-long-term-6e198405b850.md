# 比特币

投资比特币是新人涉足加密货币市场最常见的方法。它是世界上最有价值的加密货币。当比特币的价值在 2021 年底达到 68，000 美元以上时，该项目的市值超过 1 万亿美元。这导致比特币比世界上最知名的几种股票都更有价值。
