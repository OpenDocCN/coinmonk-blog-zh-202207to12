# 比特币(BTC)

对于所谓的蓝筹加密货币来说，现在可能是最关键的时刻。这些数字货币的品牌、用途和忠诚度已经非常成熟，加密货币本身必须灭亡才能彻底垮台。第一种加密货币比特币的市场估值接近 3200 亿美元，轻松胜出。
