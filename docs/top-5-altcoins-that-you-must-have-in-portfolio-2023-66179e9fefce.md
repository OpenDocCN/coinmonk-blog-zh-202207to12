# 以太坊

以太坊的可访问性和不稳定的价格波动使其成为我们 2023 年最佳购买加密货币名单的第一名。在这种情况下，“可获得”是指它在几乎所有主要的加密货币交易所交易，而“波动性”是指价格快速上涨和下跌的趋势。
