# **描述**

以太坊是一个支持智能合约的分布式应用和分布式区块链计算平台。以太网(ETH)是该平台的本机令牌，主要用于支付交易费用，并作为分散金融(DeFi)行业中某些 ERC 20 令牌的贷款担保。
