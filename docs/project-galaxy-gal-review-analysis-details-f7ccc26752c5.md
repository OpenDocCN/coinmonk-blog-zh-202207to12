**当前凭据系统的问题**
如今，凭据存储在由应用程序、政府机构和金融机构拥有和授权的封闭数据库中。

需要拥有数据或应用程序的个人将受到影响。卡拉狄加计划旨在通过使用开放和协作的凭证网络来解决这个问题。这个网络将帮助 Web3 开发者使用他们的证书来创造更好的产品和更吸引人的社区。
