超级简单的设置

使用Pathfinder和Watchtower建立一个自我更新的StarkNet节点。

  1. 克隆我的docker合成文件
git clone https://github.com/0xytoken/pathfinder-docker.git

2.输入repo并调用docker-compose up。

cd pathfinder-docker && docker-compose up

3.如果探路者和瞭望塔旋转得很漂亮，那么停止它们并再次运行命令，但这次是在分离模式(又名。背景)

docker-compose -d up

4.如果实例的端口是开放的，那么现在可以使用JSON-RPC端点在端口9545上查询节点的状态。

你可以在这里随意尝试一下。
