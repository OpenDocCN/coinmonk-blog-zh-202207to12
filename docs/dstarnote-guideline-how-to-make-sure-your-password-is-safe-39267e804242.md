传统的笔记应用程序是基于云的，由一堆集中式服务器托管，由某些实体(通常是大公司)运营和控制。这种基于云的笔记应用程序不可信，因为即使有冗余备份，数据也可能因硬件问题或其他问题而丢失。任何可以访问数据库的人都可以删除或泄露你所有的笔记，这显然是一场灾难。
