比特币正在崛起已经不是什么秘密了。过去一年，这种加密货币的价值飙升，并且没有放缓的迹象。亿万富翁投资者沃伦·巴菲特说，你唯一应该投资的时候是你愿意全部购买的时候。这条建议违背了分散投资的传统智慧，并让我提出了一个问题:我应该全押比特币吗？
