# 网飞(NFLX)

本周寻找更高风险股票的投资者可以考虑流动巨头网飞(纳斯达克代码:NFLX)。当然，我可以理解为什么不愿意来这里。举例来说，市场损失是残酷的。自今年年初以来，NFLX 已经损失了近 51%的股票价值。此外，订阅量下降表明人们对家庭娱乐选择的兴趣下降。
