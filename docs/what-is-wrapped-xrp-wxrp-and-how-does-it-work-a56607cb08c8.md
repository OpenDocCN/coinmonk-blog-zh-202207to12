**包装 XRP (wXRP)是一种与 XRP (XRP)挂钩的加密资产，可用于区块链，而不是 Ripple 的本地 XRP 总账。**包装 XRP 是一种令牌，旨在解决跨不同网络的资产可移植性问题。
