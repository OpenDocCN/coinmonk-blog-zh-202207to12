# 阿尔格兰德(ALGO)

Algorand (ALGO)是一个加密货币和区块链平台，在一个两层、分散的网络上运行。由于区块链是开源的，任何人都可以查看代码并对平台进行更改。ALGO (ALGO)的目标是提供一个既可扩展又高效的平台，允许快速执行交易。
