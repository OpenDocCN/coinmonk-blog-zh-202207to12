# FTX 首席执行官 Sam Bankman-Fried

萨姆·班克曼-弗里德是加密货币衍生品交易所 FTX 的首席执行官。他也是这个领域最具创新精神的人之一，总是想出新的方法来发展他的业务。在短短的几年里，Sam Bankman-Fried 已经将 FTX 建成了世界上最大、最成功的加密货币交易所之一。他还开发了许多独特的产品，使 FTX 成为该行业的领导者。
